One step to go live

Add your
Google Earth key

This experience flies through Google's real, photorealistic 3D Earth. That needs a free Google Maps API key — paste yours into experience.html where it says GOOGLE_MAPS_API_KEY.

  1. Go to Google Cloud → Maps and create / pick a project.
  2. Enable the Map Tiles API.
  3. Create an API key (Credentials → Create credentials → API key).
  4. Paste it into experience.html at const GOOGLE_MAPS_API_KEY = "…" and reload.
